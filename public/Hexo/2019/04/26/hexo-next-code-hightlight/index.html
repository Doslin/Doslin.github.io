hexo代码显示深度定制化

​

本文主要记录一下，修改Hexo下语法高亮的各种用法，修改后的语法可以快速设置多种样式，主要功能如：

  • 设置语法语言

  • 设置标题

  • 是否显示行号

  • 设置起始行号

  • 选定行号标记

  • 设置代码添加删除标记 （新增）

  • 设置跳转超级链接
